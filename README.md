## Sistema de Color

| Token | Valor | Uso |
|---|---|---|
| Dark background | `#070213` | Fondo principal |
| Light background | `#FFFFFF` | Variante light |
| Paths sobre dark | `#FFFFFF` | Ícono y texto sobre `#070213` |
| Paths sobre light | `#070213` | Ícono y texto sobre blanco |
